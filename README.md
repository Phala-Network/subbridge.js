# subbridge.js - SubBridge JavaScript SDK

## Installation

### Node

```bash
yarn add subbridge.js
```

### In the Browser

Use prebuilt target `dist/subbridge.min.js`, or build from source code:

```bash
yarn build
```

## Usage

```js
// In Node.js
const SubBridge = require(`subbridge.js`);
```
